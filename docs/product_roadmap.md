# WealthPulse Product Roadmap (Draft)

## Phase 1: MVP Polish (Weeks 1-4)
Goal: Make the app effortless, trustworthy, and launch-ready for first-time users.

1. Guided onboarding (5-step wizard)
   - Business vs personal entities
   - Entity setup (trust, company, personal)
   - Asset input (quick add presets)
   - Liabilities
   - Goals and milestones

2. Trust and transparency
   - Clear "Where your data lives" panel
   - Local vs cloud labeling
   - Backups and export instructions

3. AI valuation trust
   - "Why this estimate?" explainers
   - Manual override with confidence slider
   - Clear source labels for live pricing

4. UI consistency and polish
   - Contrast sweep for light/dark themes
   - Tighten spacing and typography
   - Remove any raw HTML render artifacts

5. Support and onboarding help
   - Built-in Quick Guide tab
   - Visible feedback button
   - FAQ for common issues

## Phase 2: Premium Features (Weeks 5-12)
Goal: Add paid, high-value functionality for small business owners.

1. Entity mapping visualization
   - Ownership graph: person -> trust -> business -> assets

2. Scenario planner
   - "What if I transfer this asset to a trust?"
   - "What if I buy a property in 18 months?"

3. Business-owner toolkit
   - Business vs personal balance overview
   - Exportable summary for accountants

4. Market insights upgrades
   - Premium live data sources
   - Alerts and signals for bullion and equities

## Phase 3: Community & Marketplace (Optional Tier)
Goal: Increase engagement and paid conversions.

1. Verified seller profiles
2. Live auctions with reserve and buy-now
3. Buyer/seller moderation tools
4. Messaging and transaction disclaimers

## Success Metrics
- Activation rate: 60% complete onboarding
- D1 retention: 35%
- Paid conversion: 8-12%
- Support volume: < 1 ticket per 30 users

## Release Targets
- MVP polish: 4 weeks
- Premium feature set: 8-12 weeks
- Marketplace improvements: 12-16 weeks
